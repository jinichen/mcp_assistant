# User Manual - Multimodal Chat Application

Welcome to the Multimodal Chat Application! This comprehensive guide will help you navigate the features and capabilities of our advanced chat system that supports text, images, and AI-powered tools.

## Getting Started

### System Requirements

- Modern web browser (Chrome, Firefox, Safari, Edge)
- Internet connection
- Device with camera (optional, for capturing images)

### Accessing the Application

1. Open your web browser
2. Navigate to the application URL
3. The interface will load automatically

## Main Interface

The chat interface consists of:

- **Chat History Panel**: Displays all messages in the conversation
- **Input Area**: Where you type your messages
- **Model Selection**: Choose which AI model to use
- **Image Upload**: Add images to your messages
- **Tool Access**: Access to various AI-powered tools

## Sending Messages

### Text Messages

1. Type your message in the input area at the bottom of the screen
2. Press Enter or click the Send button (paper airplane icon)
3. Your message will appear in the chat history and the AI will respond

### Including Images

1. Click the Image Upload button (image icon) next to the text input
2. Either:
   - Click the file icon to browse for images on your device
   - Drag and drop images directly into the upload area
3. Selected images will appear as previews
4. You can remove images by clicking the X on each preview
5. Type any text you want to include with the images (optional)
6. Click the "Send [number] image(s)" button to send your message with the attached images
7. Images will appear in the chat history along with any text you included

## Working with AI Models

### Selecting a Model

1. Use the model dropdown menu at the top of the chat interface
2. Available models are grouped by provider (Google, OpenAI, NVIDIA, Anthropic)
3. Select your preferred model from the list
4. The system will use your selected model for all subsequent messages

### Model Capabilities

Different models have different capabilities:

- **All models**: Support text conversations
- **Multimodal models**: Support both text and image processing
- **Tool-enabled models**: Can use external tools to perform tasks

The system automatically detects what each model supports and adjusts accordingly.

## Using Sessions

### Creating a New Session

1. Click the "New Chat" button to start a fresh conversation
2. The previous conversation will be cleared

### Saving a Session

1. Click the Save button (floppy disk icon)
2. Enter a title for your session
3. Click "Save Session"
4. The conversation will be saved for future access

### Loading a Saved Session

1. Click the Sessions button (list icon)
2. Select a saved session from the list
3. The conversation history will be loaded

### Deleting a Session

1. Click the Sessions button (list icon)
2. Find the session you want to delete
3. Click the Delete button (trash icon) next to that session
4. Confirm the deletion

## Troubleshooting

### Common Issues

#### Image Upload Problems

- **Issue**: Images won't upload
- **Solution**: Check that your images are in a supported format (JPG, PNG, GIF, WebP) and under 10MB

#### Model Errors

- **Issue**: "This model doesn't support images"
- **Solution**: Switch to a multimodal model (like GPT-4 Vision, Gemini Pro Vision)

#### Slow Responses

- **Issue**: AI takes too long to respond
- **Solution**: Try switching to a smaller, faster model or simplify your query

#### Connection Errors

- **Issue**: "Failed to connect to server"
- **Solution**: Check your internet connection and refresh the page

### Getting Support

If you encounter persistent issues:

1. Check the error message for specific details
2. Refresh the application
3. Try a different browser
4. Contact support with details about the issue

## Best Practices

### Effective Prompting

For best results with the AI:

- Be specific and clear in your questions
- Provide context for your queries
- Break complex tasks into smaller steps
- Use appropriate models for specific tasks (e.g., vision models for image analysis)

### Working with Images

For optimal image processing:

- Use clear, high-quality images
- Ensure good lighting for photographs
- Crop images to focus on the relevant content
- Include specific questions about the image

### Security Considerations

For your security:

- Don't upload sensitive personal information in images
- Be aware that conversations may be stored for service improvement
- Use secure and private networks when accessing the application

## Keyboard Shortcuts

- **Enter**: Send message
- **Shift + Enter**: Add new line in message
- **Esc**: Clear current input
- **Ctrl/Cmd + N**: New chat
- **Ctrl/Cmd + S**: Save session
